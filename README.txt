API Tviso
La API de Tviso es un servicio web gratuito para obtener información de los medios. Todo el contenido y las imágenes en el sitio son contribuidos y mantenidos por sus usuarios. Actualmente, esta API se encuentra en estado Beta, por lo que algunas peticiones no funcionan correctamente. Esta API consta de dos capas bien diferenciadas:
-Capa de medios: Permite obtener todos los datos relativos de los medios (películas, series, shows de televisión, etc). Esta es la capa que se ha utilizado en esta aplicación.
-Capa Social: Le permite al usuario tener su propia colección multimedia, comentar medios, calificar medios, recomendar contenido a otros usuarios, entre otras actividades. Esta capa no se ha utilizado en esta aplicación ya que era necesario que el usuario que maneje la app este registrado en la página Web de TVISO.
